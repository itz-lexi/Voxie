Easy to try

QUICK SETUP

From download to chatbox

  1. Install Voxie or download the portable ZIP.
  2. In VRChat, enable Action Menu > OSC > Enabled.
  3. Download the local base.en model from inside Voxie.
  4. Select a microphone and press your shortcut or the recording button.
